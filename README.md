## Projects

### Session 1 - Youtube Controller

A Chrome extension that allows you to control YouTube playback from any tab with a compact control panel.

**Features:**
- 🎬 Video preview with thumbnail
- ⏯️ Play/Pause controls
- ⏭️ Next/Previous video navigation
- ⏪⏩ Skip backward/forward (10 seconds)
- 📊 Seekable progress bar
- 🔊 Volume controls (up/down/mute)
- 🚀 Works from any Chrome tab

**[View Full Documentation](Session%201/README.md)**

### Session 2 - Hotel Comparison Assistant

An AI-powered Chrome extension that helps you compare hotels across multiple booking websites using Google Gemini.

**Features:**
- 🏨 Extract hotel data from ANY website (Booking.com, MakeMyTrip, Hotels.com, etc.)
- 🤖 AI-powered automatic data extraction (name, price, rating, amenities, reviews)
- 📍 Distance calculation from your reference location
- 🔄 Compare hotels from different websites side-by-side
- 🎯 Custom comparison criteria with AI recommendations
- 📊 Beautiful formatted comparison results
- 🖼️ Hotel images with smart extraction
- ⚡ Works across all major hotel booking platforms

**Tech Stack:**
- Chrome Extension (Manifest V3)
- Python Flask Backend
- Google Gemini AI API
- BeautifulSoup for HTML parsing
- GeoPy for distance calculation

**[View Full Documentation](Session%202/README.md)**

---

### Session 3 - AI Game Character Creator

An intelligent Chrome extension that creates complete game characters using Google Gemini 3 AI agents with advanced function calling and thought signatures.

**Features:**
- 🤖 AI Agent Orchestration with 6 specialized character design tools
- 🧠 Reasoning Chain Visualization - See the AI think step-by-step!
- 🎨 AI-Generated Character Portraits (gemini-2.5-flash-image)
- 📊 Balanced attributes, backstory, abilities, appearance, equipment, personality
- 💾 Save/Load characters with full reasoning chains
- 📤 Export to JSON
- ⚡ Creates complete characters in ~10-15 seconds
- 🔍 Transparent AI reasoning with thought signatures

**Tech Stack:**
- Chrome Extension (Side Panel API)
- Python Flask Backend
- Google Gemini 3 Flash Preview (AI Agent)
- gemini-2.5-flash-image (Nano Banana)
- Function Calling with Thought Signatures
- google-genai SDK v1.73.1

**[View Full Documentation](Session%203/README.md)**
